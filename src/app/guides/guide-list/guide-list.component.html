READ ME

The application is based on the framework of Angular. Both front-end pages and back-end server are developed, in order to keep data consistent, even if refresh or restart. It is fully functional in terms of CRUD (Add, Display, Update, Delete). To exchange data between two lists, users can complete that via either drag and drop the item individually, or click to select all items at one time, or select all items and then uncheck the not desired ones. All three ways are doable in the application.

The main libraries and tech used for building this application are as follow:

  1. From Angular library "Materal", the @angular/cdk/drag-drop module is used for achieving the Drag and Drop functionality to swap data between lists.

  2. Angular Lazy Loading is used in the two routes in the program to navigate url links.

  3. Angularfire2 and firebase are used as remote data server. It consistently fetches and saves data with front-end users' input.

  4. Bootstrap 4 is used to keep web pages responsive.
